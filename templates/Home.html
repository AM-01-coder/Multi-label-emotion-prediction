Multi-label Emotion Prediction System

Project Aim

The main aim of Multi-label Emotion Prediction System is to predict one or more (most dominating) Emotions in a given English text from a pre-defined set of Emotions (Anger, Anticipation, Disgust, Fear, Joy, Love, Optimism, Pessimism, Sadness, Surprise, Trust or Neutral).


Input: An English Text

Output: Emotion(s)


Example - Input and Output

Input: The best medicine of the world is: Love and Respect the Humanity to Please Allah.

Output: Joy, Love, Optimism
